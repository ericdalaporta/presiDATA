## Formato das imagens
- Formato: JPG ou PNG
- Tamanho recomendado: 400x400 pixels (quadrado)
- Qualidade: Alta resolução para boa visualização

## Fontes sugeridas para as imagens
- Site oficial da Presidência da República
- Arquivo Nacional
- Biblioteca Nacional
- Museus históricos
- Wikimedia Commons (verificar licenças)

## Imagem padrão
Caso não encontre alguma foto específica, será usado um placeholder padrão.
